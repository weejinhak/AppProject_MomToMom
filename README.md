# AppProject_MomToMom
모유가 걱정인 엄마들을 위한 앱
수유실을 통해 안전한 만남을 가지세요!

## 🔍 프로젝트 소개

갓 태어난 아이들에게 불가피한 사정으로 모유 수유를 못하는 엄마들에게 전국의 정확한 수유실 위치를 바탕으로
기증자에 대한 정보를 확인하고 기증 받을 수 있게 합니다. 또한, 모유 수유에 대한 상식과 정확한 가이드를 제공하여
모유 수유를 원하는 엄마들을 돕는 서비스를 해보고자 합니다.
